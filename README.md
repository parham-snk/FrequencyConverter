## Dependencies
- ffmpeg
- electron
- electron-builder

## install ffmpeg app for your platform
for run this app you need to install ffmpeg.

[download from original website !](https://www.ffmpeg.org/)
## install packages
first install the packages!

``` bash
npm i --save
```

## build

``` bash
npm run build
```
and **wait** build complete.



enjoy !
